# Project: Simple code & AI agents

Estimate the age of a person based on a name and country code (optional).

**Backend**

Uses API from [Agify](https://agify.io/documentation) to estimate the age of a person based on a name. As naming conventions can rely on demogrpahics, it is optional to add a country code/id. A full list of available country IDs: https://agify.io/our-data
